# 🎓 Student360 – Student Performance Dashboard

## 📊 Overview

**Student360** is an interactive data analytics dashboard designed to analyze student performance based on behavioral, academic, and lifestyle factors. It provides actionable insights into how habits like studying, gaming, sleep, and attendance influence academic outcomes.

---

## 🚀 Features

* 📈 **Behavioral Insights**

  * Study hours vs performance
  * Gaming addiction vs performance
  * Sleep patterns vs grades

* 📚 **Academic Analysis**

  * Attendance impact on grades
  * Study efficiency scoring
  * High-performing student identification

* 🎮 **Gaming Analysis**

  * Gaming hours vs grades
  * Gaming genre impact (FPS, RPG, Casual)
  * Addiction score correlation

* 👥 **Demographic Insights**

  * Gender-based performance comparison
  * Lifestyle and stress-level impact

---

## 🧰 Tech Stack

* **Power BI** – Dashboard & Data Visualization
* **SQL** – Data querying and transformation
* **Excel / CSV** – Dataset handling
* **Power Query** – Data cleaning and preprocessing

---

## 🔍 Key Insights

* 📚 More study hours generally lead to better academic performance.
* 💤 Proper sleep (6–8 hours) correlates with higher grades.
* 🎮 Excessive gaming and higher addiction scores negatively impact performance.
* 📊 Attendance has a strong positive relationship with grades.
* ⚖️ Balanced lifestyle = optimal academic success.
